01 · Apa itu MCP

Satu protokol. AI apa pun terhubung ke akun Anda.

MCP (Model Context Protocol) adalah standar terbuka dari Anthropic yang memungkinkan asisten AI terhubung ke sistem eksternal. Kami menyediakan server untuk setiap jenis instrumen — Anda membuat token dan menempelkannya ke asisten Anda.

Analisis pasar dengan bahasa sehari-hari

Tanyakan kepada AI tentang kuotasi langsung, candle historis, spread, volume — dengan kata-kata Anda sendiri, di keempat pasar sekaligus.

Lihat portofolio Anda di chat

Posisi terbuka, order tertunda, riwayat trading, total ekuitas — semua melalui percakapan biasa. Tanpa berpindah jendela.

Kontrol di tingkat token

Setiap token adalah integrasi terpisah dengan masa berlaku tersendiri. Lihat semua yang aktif di Pengaturan → Integrasi AI dan cabut salah satunya dengan satu klik.

03 · Izin token

Baca secara default. Trading — opsional, per pasar.

Setiap token memiliki cakupannya sendiri. Akses baca ke keempat server diaktifkan secara otomatis dan tidak dapat dimatikan — AI membutuhkannya hanya untuk "melihat" data Anda. Hak untuk membuka dan menutup trading diberikan secara eksplisit — terpisah untuk setiap pasar, saat Anda membuat token.

Baca — tidak dapat dimatikan

Tanpa akses baca, AI tidak berguna — ia tidak dapat menampilkan saldo atau harga Anda. Itulah sebabnya cakupan baca diaktifkan secara otomatis di keempat server. Ini adalah tingkat yang paling aman: AI hanya melihat, tidak pernah menyentuh.

Trading — presisi, per kasus penggunaan

Ingin AI membantu trading options saja? Centang Trade · Digital Options dan tidak ada yang lain. Margin CFD/Crypto/Forex tetap hanya-baca — sekalipun AI mencoba, server akan mengembalikan 403 Forbidden.

Cakupan dapat diubah tanpa membuat ulang token

Ingin menghapus izin trading dari token yang sudah digunakan? Buka Settings → AI integrations, temukan token tersebut, hapus centang pasar — izin langsung dihapus. Nilai token itu sendiri tidak berubah, jadi tidak perlu memperbarui konfigurasi asisten Anda.

04 · Cara menghubungkan

Tiga langkah. Tanpa kode, tanpa terminal.

MCP kami berjalan melalui streamable-http — tanpa lingkungan Python, tanpa virtual venv, tanpa repo untuk di-clone. Hanya URL + token Bearer yang Anda buat dengan satu klik.

1

Buat token

Di aplikasi seluler atau di web:

  1. Buka Pengaturan → Integrasi AI →
  2. Ketuk + Tambah integrasi
  3. Beri token nama dan masa kedaluwarsa
  4. Centang pasar mana yang diizinkan untuk trading (baca selalu aktif). Detail — bagian 03 di atas
  5. Salin token segera — setelah Anda menutup layar, token tidak akan ditampilkan lagi
Buka Integrasi AI →
2

Pilih asisten Anda

Dukungan MCP native tersedia di:

  1. Claude Desktop / Claude Code
  2. Cursor / Windsurf / Zed
  3. OpenAI Codex / ChatGPT Desktop
  4. Cline / Continue / Goose / LibreChat
3

Tempel konfigurasi

Salin cuplikan untuk asisten Anda. Masing-masing membutuhkan tiga hal:

  1. Server URL
  2. Authorization: Bearer + token Anda
  3. Mulai ulang asisten Anda
05 · Konfigurasi siap pakai

Pilih asisten Anda — konfigurasinya sudah siap.

06 · Keamanan

Token Anda, tanggung jawab Anda. Pengaman sudah terpasang.

MCP memberikan AI kunci ke akun trading Anda. Karena itu kami — dan Anda — mengawasi batasan bersama-sama.

Token berada di header, bukan di chat

Token Bearer dilampirkan ke header HTTP pada koneksi asisten ↔ server. AI tidak melihatnya di badan pesan dan tidak dapat mengirimkannya secara tidak sengaja.

Jangan dibagikan. Jangan di-commit.

Token = akses ke akun trading Anda. Jangan pernah mempublikasikannya di git, tangkapan layar, chat, atau gist. Jika Anda mencurigai adanya kebocoran — buka Pengaturan → Integrasi AI dan tekan Cabut di samping token tersebut. Semua permintaan yang menggunakannya langsung berhenti berfungsi.

Kedaluwarsa dan rotasi

Anda menetapkan masa kedaluwarsa saat pembuatan. Setelah tanggal tersebut, token berhenti berfungsi secara otomatis. Kami menyarankan menyimpan token terpisah per integrasi — dengan begitu Anda hanya perlu mencabut yang diperlukan.

Mulai dari yang kecil

AI itu canggih tetapi tidak sempurna. Ia bisa memilih instrumen, ukuran, atau arah yang salah. Sebelum memercayakan ukuran kepada model — telusuri skenario secara manual dan periksa kembali sarannya. Tanggung jawab atas setiap trading ada pada Anda.

08 · Pertanyaan yang sering diajukan

FAQ

Apakah saya perlu menginstal sesuatu di komputer saya?

Tidak. Server MCP kami di-host oleh kami — tanpa repo untuk di-clone, tanpa Python atau venv untuk diinstal. Cukup URL + token Bearer di konfigurasi asisten AI Anda.

Apakah satu token benar-benar berfungsi di keempat server?

Ya. Buat satu token Bearer di Settings → AI integrations dan gunakan di header Authorization untuk keempat endpoint: digital-options, marginal-cfd, marginal-crypto, marginal-forex.

Bagaimana jika saya hanya ingin menghubungkan satu pasar, misalnya Forex?

Hapus blok lainnya dari konfigurasi — simpan hanya iq-margin-forex. AI hanya akan melihat tool dari server tersebut dan tidak akan dapat menyentuh, misalnya, options secara tidak sengaja.

Apa yang sebenarnya dapat dilakukan AI melalui MCP?

Masing-masing server kami menyediakan daftar tool-nya sendiri — Anda dapat melihat set lengkapnya tepat setelah terhubung dengan menanyakan kepada asisten: "Tool IQ Option apa saja yang Anda miliki?". Secara keseluruhan ini meliputi pembacaan data pasar, portofolio, dan trading, ditambah metode untuk membuka/menutup posisi. Metode mana yang tersedia untuk token tertentu ditentukan oleh cakupannya (baca + opsional trading per pasar).

Bagaimana cara memastikan AI tidak dapat melakukan trading?

Saat membuat token di Settings → AI integrations cukup jangan centang satu pun dari keempat kotak centang "Trading · …". Baca tetap aktif (tidak dapat dimatikan — AI membutuhkannya), dan setiap upaya untuk memanggil tool trading akan mengembalikan 403 Forbidden. Ini adalah penyiapan paling aman untuk kasus penggunaan riset.

Bisakah saya mengizinkan trading hanya pada satu instrumen?

Ya. Misalnya, centang hanya Trade · Margin Crypto — AI akan dapat membuka/menutup posisi kripto, tetapi options, CFD, dan forex tetap hanya-baca. Berguna saat Anda menguji penyiapan AI pada pasar yang Anda kuasai.

Bagaimana jika saya ingin menghapus izin trading dari token yang sudah ada?

Buka Settings → AI integrations, temukan token tersebut dan hapus centang kotak yang relevan (mis. Trade · Margin CFD). Izin langsung dihapus — server akan mulai mengembalikan 403 Forbidden pada panggilan trading untuk pasar tersebut. Nilai token itu sendiri tidak berubah, jadi tidak perlu mengutak-atik konfigurasi asisten Anda.

Bagaimana jika token saya bocor?

Buka Settings → AI integrations, temukan token yang disusupi dan tekan Cabut. Semua permintaan yang menggunakannya langsung berhenti berfungsi. Buat yang baru dan perbarui konfigurasi asisten.

Apa saja batasan untuk token dan server?

Satu akun dapat memiliki beberapa token aktif sekaligus (terlihat dalam daftar Token aktif). Satu token berfungsi di keempat server. Jika Anda menginginkan kunci terpisah per integrasi agar lebih mudah dicabut — buat beberapa dan beri nama yang bermakna.

Asisten saya tidak melihat tool setelah penyiapan. Apa yang harus dilakukan?

Daftar periksa: (1) mulai ulang asisten sepenuhnya, (2) pastikan token disalin tanpa spasi tambahan di awal/akhir, (3) periksa bahwa URL tidak memiliki garis miring di akhir dan sesuai dengan langganan Anda, (4) pastikan header Authorization dimulai persis dengan Bearer (perhatikan spasinya), (5) periksa log asisten — biasanya ini adalah kesalahan sintaks JSON pada konfigurasi.

Asisten mana yang paling mudah disiapkan dari awal?

Jika Anda bukan developer — Claude Desktop atau Cherry Studio: keduanya memiliki UI yang menangani konfigurasi untuk Anda. Jika Anda terbiasa dengan kode — Cursor atau Claude Code akan terasa lebih alami.
