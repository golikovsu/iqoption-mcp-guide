01 · ما هو MCP

بروتوكول واحد. أي ذكاء اصطناعي يتواصل مع حسابك.

MCP (Model Context Protocol) هو معيار مفتوح من Anthropic يتصل من خلاله مساعدو الذكاء الاصطناعي بالأنظمة الخارجية. نستضيف خوادم لكل نوع من الأدوات — تنشئ رمزاً وتلصقه في مساعدك.

⚡

حلّل الأسواق بلغة بسيطة

اسأل الذكاء الاصطناعي عن الأسعار المباشرة والشموع التاريخية والفروق السعرية والأحجام — بكلماتك الخاصة، عبر الأسواق الأربعة دفعةً واحدة.

📊

اطّلع على محفظتك في المحادثة

المراكز المفتوحة، والأوامر المعلّقة، وسجل الصفقات، وإجمالي حقوق الملكية — كل ذلك عبر محادثة بسيطة. دون تبديل النوافذ.

🛡️

تحكّم على مستوى الرمز

كل رمز هو تكامل منفصل بمدة صلاحية خاصة به. اطّلع على جميع الرموز النشطة في الإعدادات ← تكاملات الذكاء الاصطناعي وألغِ أيّاً منها بنقرة واحدة.

03 · أذونات الرمز

القراءة افتراضية. التداول — اختياري، لكل سوق على حدة.

لكل رمز نطاقه الخاص. صلاحية القراءة لجميع الخوادم الأربعة مفعّلة تلقائياً ولا يمكن إيقافها — يحتاجها الذكاء الاصطناعي فقط كي "يرى" بياناتك. أما حق فتح الصفقات وإغلاقها فيُمنح صراحةً — بشكل منفصل لكل سوق، عند إنشائك للرمز.

🟢 القراءة — لا يمكن إيقافها

بدون صلاحية القراءة يصبح الذكاء الاصطناعي عديم الفائدة — فلا يمكنه عرض رصيدك أو أسعارك. لهذا يُفعَّل نطاق القراءة تلقائياً عبر الخوادم الأربعة جميعاً. وهذا هو المستوى الأكثر أماناً: الذكاء الاصطناعي يطّلع فقط ولا يتدخّل أبداً.

🟠 التداول — دقيق، حسب حالة الاستخدام

أتريد أن يساعدك الذكاء الاصطناعي على تداول الخيارات فقط؟ علّم على Trade · Digital Options ولا شيء غير ذلك. تبقى Margin CFD/Crypto/Forex للقراءة فقط — وحتى لو حاول الذكاء الاصطناعي، يُعيد الخادم 403 Forbidden.

🔵 يمكن تغيير النطاق دون إعادة إنشاء الرمز

أتريد إزالة صلاحية تداول من رمز قيد الاستخدام بالفعل؟ افتح Settings → AI integrations، وابحث عن الرمز، وألغِ تحديد السوق — تُزال الصلاحية فوراً. ولا تتغيّر قيمة الرمز نفسها، فلا حاجة لتحديث إعداد مساعدك.

04 · كيفية الربط

ثلاث خطوات. بدون برمجة، وبدون طرفية.

يعمل MCP الخاص بنا عبر streamable-HTTP — بدون بيئات Python، وبدون venv افتراضية، وبدون مستودعات لاستنساخها. فقط URL + رمز Bearer تنشئه بنقرة واحدة.

1

أنشئ رمزاً

في تطبيق الجوال أو على الويب:

  1. افتح الإعدادات ← تكاملات الذكاء الاصطناعي ←
  2. اضغط على + إضافة تكامل
  3. امنح الرمز اسماً ومدة صلاحية
  4. علّم على الأسواق المسموح بها للتداول (القراءة مفعّلة دائماً). التفاصيل — في القسم 03 أعلاه
  5. انسخ الرمز فوراً — فبمجرد إغلاقك للشاشة لن يُعرض مجدداً
افتح تكاملات الذكاء الاصطناعي ←
2

اختر مساعدك

الدعم الأصلي لـ MCP متاح في:

  1. Claude Desktop / Claude Code
  2. Cursor / Windsurf / Zed
  3. OpenAI Codex / ChatGPT Desktop
  4. Cline / Continue / Goose / LibreChat
3

الصق الإعداد

انسخ المقتطف الخاص بمساعدك. يحتاج كل مساعد إلى ثلاثة أشياء:

  1. الخادم URL
  2. Authorization: Bearer + رمزك
  3. أعد تشغيل مساعدك
05 · إعدادات جاهزة

اختر مساعدك — الإعداد جاهز.

يربط كل مقتطف الخوادم الأربعة دفعةً واحدة. ما عليك سوى استبدال YOUR_TOKEN بالرمز الذي أنشأته في الخطوة 1. وإذا كنت تحتاج سوقاً واحداً فقط، فاحتفظ بتلك الكتلة فحسب واحذف الباقي.

سهل

▸ Windows

Claude Code (CLI)

سهل
الذكاء الاصطناعي العامل عبر الطرفية من Anthropic. أمر صدفة واحد لكل خادم. بدون ملف إعداد — كل شيء عبر CLI.
الطرفية · إضافة 4 خوادم 
# Digital Options
claude mcp add --transport http iq-digital-options \
  https://digital-options.mcp.iqoption.com \
  --header "Authorization: Bearer YOUR_TOKEN"

# Margin CFD
claude mcp add --transport http iq-margin-cfd \
  https://marginal-cfd.mcp.iqoption.com \
  --header "Authorization: Bearer YOUR_TOKEN"

# Margin Crypto
claude mcp add --transport http iq-margin-crypto \
  https://marginal-crypto.mcp.iqoption.com \
  --header "Authorization: Bearer YOUR_TOKEN"

# Margin Forex
claude mcp add --transport http iq-margin-forex \
  https://marginal-forex.mcp.iqoption.com \
  --header "Authorization: Bearer YOUR_TOKEN"
✓
تحقّق. نفّذ claude mcp list — من المفترض أن ترى 4 أسطر بحالة اتصال ✓.

06 · الأمان

رمزك، ومسؤوليتك. حواجز الأمان مدمجة.

يمنح MCP الذكاءَ الاصطناعي مفاتيح حساب التداول الخاص بك. لذا نراقب الحدود معاً — نحن وأنت.

🔐

الرمز يعيش في الترويسات، لا في المحادثة

يُرفق رمز Bearer بترويسة HTTP على اتصال المساعد ↔ الخادم. لا يراه الذكاء الاصطناعي في متن الرسالة ولا يمكنه إرساله خارجاً عن طريق الخطأ.

⚠

لا تشاركه. لا تضمّنه في الكود.

الرمز = الوصول إلى حساب التداول الخاص بك. لا تنشره أبداً في git أو لقطات الشاشة أو المحادثات أو gists. وإذا اشتبهت في تسريبه — فافتح الإعدادات ← تكاملات الذكاء الاصطناعي واضغط على إلغاء بجوار ذلك الرمز. ستتوقف جميع الطلبات التي تستخدمه عن العمل فوراً.

08 · الأسئلة الشائعة

الأسئلة الشائعة

هل يلزمني تثبيت أي شيء على حاسوبي؟

لا. نستضيف خوادم MCP الخاصة بنا بأنفسنا — بدون مستودعات لاستنساخها، وبدون Python أو venv لتثبيتها. فقط URL + رمز Bearer في إعداد مساعد الذكاء الاصطناعي الخاص بك.

هل يعمل رمز واحد فعلاً على الخوادم الأربعة جميعاً؟

نعم. أنشئ رمز Bearer واحداً في Settings → AI integrations واستخدمه في ترويسة Authorization لجميع نقاط النهاية الأربع: digital-options وmarginal-cfd وmarginal-crypto وmarginal-forex.

ماذا لو أردت ربط سوق واحد فقط، مثل الفوركس؟

احذف الكتل الأخرى من الإعداد — واحتفظ فقط بـ iq-margin-forex. سيرى الذكاء الاصطناعي أدوات ذلك الخادم فقط ولن يتمكّن من المساس، مثلاً، بالخيارات عن طريق الخطأ.

ما الذي يستطيع الذكاء الاصطناعي فعله فعلاً عبر MCP؟

يكشف كل خادم من خوادمنا قائمة أدواته الخاصة — ويمكنك الاطّلاع على المجموعة الكاملة فور الربط بسؤال المساعد: "ما أدوات IQ Option المتاحة لديك؟". وإجمالاً هذا يشمل قراءة بيانات السوق والمحفظة والصفقات، إضافةً إلى طرق فتح المراكز وإغلاقها. والطرق المتاحة لرمز معيّن يحدّدها نطاقه (قراءة + تداول اختياري لكل سوق).

كيف أتأكّد من أن الذكاء الاصطناعي لا يستطيع التداول؟

عند إنشاء الرمز في Settings → AI integrations ببساطة لا تعلّم على أي من خانات "تداول · …" الأربع. تبقى القراءة مفعّلة (لا يمكن إيقافها — يحتاجها الذكاء الاصطناعي)، وأي محاولة لاستدعاء أداة تداول ستُعيد 403 Forbidden. وهذا هو الإعداد الأكثر أماناً لحالات استخدام البحث.

هل يمكنني السماح بالتداول على أداة واحدة فقط؟

نعم. على سبيل المثال، علّم فقط على Trade · Margin Crypto — سيتمكّن الذكاء الاصطناعي من فتح مراكز العملات المشفرة وإغلاقها، لكن تبقى الخيارات وCFD والفوركس للقراءة فقط. مفيد عندما تختبر إعداد الذكاء الاصطناعي على سوق تشعر بالارتياح تجاهه.

ماذا لو أردت إزالة صلاحية تداول من رمز موجود؟

افتح Settings → AI integrations، وابحث عن الرمز وألغِ تحديد الخانة المعنية (مثل Trade · Margin CFD). تُزال الصلاحية فوراً — وسيبدأ الخادم بإعادة 403 Forbidden على استدعاءات التداول لذلك السوق. ولا تتغيّر قيمة الرمز نفسها، فلا حاجة للمساس بإعداد مساعدك.

ماذا لو تسرّب رمزي؟

انتقل إلى Settings → AI integrations، وابحث عن الرمز المخترق واضغط على إلغاء. ستتوقف جميع الطلبات التي تستخدمه عن العمل فوراً. أنشئ رمزاً جديداً وحدّث إعداد المساعد.

ما حدود الرموز والخوادم؟

يمكن أن يكون لحساب واحد عدّة رموز نشطة في آنٍ واحد (تظهر في قائمة الرموز النشطة). ويعمل الرمز الواحد عبر الخوادم الأربعة جميعاً. وإذا أردت مفاتيح منفصلة لكل تكامل لتسهيل الإلغاء — فأنشئ عدّة رموز وامنحها أسماءً ذات دلالة.

مساعدي لا يرى الأدوات بعد الإعداد. ماذا الآن؟

قائمة التحقق: (1) أعد تشغيل المساعد بالكامل، (2) تأكّد من نسخ الرمز دون مسافات بيضاء زائدة في بدايته أو نهايته، (3) تحقّق من أن عنوان URL لا يحتوي على شرطة مائلة لاحقة وأنه يطابق اشتراكك، (4) تأكّد من أن ترويسة Authorization تبدأ تماماً بـ Bearer (لاحظ المسافة)، (5) راجع سجلات المساعد — عادةً ما يكون خطأ في صياغة JSON ضمن الإعداد.

أي مساعد أسهل في الإعداد من الصفر؟

إن لم تكن مطوّراً — فاختر Claude Desktop أو Cherry Studio: كلاهما يوفّر واجهة تتولّى الإعداد نيابةً عنك. وإن كنت تعيش في الكود — فسيبدو Cursor أو Claude Code أكثر طبيعيةً لك.

هل استخدام MCP مجاني؟

في الوقت الحالي — نعم، مجاني لجميع عملاء IQ Option أصحاب الحسابات النشطة. أنت تدفع فقط مقابل اشتراك مساعد الذكاء الاصطناعي الخاص بك (Claude Pro / ChatGPT Plus، إلخ).
