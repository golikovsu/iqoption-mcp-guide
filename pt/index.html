01 · O que é o MCP

Um protocolo. Qualquer IA conversa com sua conta.

O MCP (Model Context Protocol) é um padrão aberto da Anthropic pelo qual assistentes de IA se conectam a sistemas externos. Hospedamos servidores para cada tipo de instrumento — você cria um token e o cola no seu assistente.

⚡

Analise mercados em linguagem natural

Pergunte à IA sobre cotações ao vivo, candles históricos, spreads, volumes — com suas próprias palavras, nos 4 mercados ao mesmo tempo.

📊

Veja seu portfólio no chat

Posições abertas, ordens pendentes, histórico de operações, patrimônio total — tudo por meio de uma conversa simples. Sem trocar de janela.

🛡️

Controle no nível do token

Cada token é uma integração separada com seu próprio tempo de vida. Veja todas as ativas em Configurações → Integrações de IA e revogue qualquer uma delas com um clique.

03 · Permissões do token

Leitura por padrão. Operar — opcional, por mercado.

Cada token tem seu próprio escopo. O acesso de leitura aos 4 servidores é ativado automaticamente e não pode ser desativado — a IA precisa dele apenas para "ver" seus dados. O direito de abrir e fechar operações é concedido explicitamente — separadamente para cada mercado, no momento em que você cria o token.

🟢 Leitura — não pode ser desativada

Sem acesso de leitura a IA é inútil — ela não consegue mostrar seu saldo ou preços. Por isso o escopo de leitura é ativado automaticamente em todos os 4 servidores. Este é o nível mais seguro: a IA apenas observa, nunca toca.

🟠 Operar — cirúrgico, por caso de uso

Quer que a IA ajude a operar apenas opções? Marque Trade · Digital Options e nada mais. CFD/Cripto/Forex com Margem permanecem somente leitura — mesmo que a IA tente, o servidor retorna 403 Forbidden.

🔵 O escopo pode ser alterado sem recriar o token

Quer remover uma permissão de operar de um token que já está em uso? Abra Settings → AI integrations, encontre o token, desmarque o mercado — a permissão é removida instantaneamente. O valor do token em si não muda, então não é preciso atualizar a configuração do seu assistente.

04 · Como conectar

Três passos. Sem código, sem terminal.

Nosso MCP roda sobre streamable-HTTP — sem ambientes Python, sem venv virtual, sem repositórios para clonar. Apenas uma URL + token Bearer que você cria em um clique.

1

Crie um token

No aplicativo móvel ou na web:

  1. Abra Configurações → Integrações de IA →
  2. Toque em + Adicionar integração
  3. Dê ao token um nome e uma validade
  4. Marque quais mercados são permitidos para operar (a leitura está sempre ativa). Detalhes — seção 03 acima
  5. Copie o token imediatamente — depois que você fechar a tela ele não será exibido novamente
Abra Integrações de IA →
2

Escolha seu assistente

Suporte nativo a MCP está disponível em:

  1. Claude Desktop / Claude Code
  2. Cursor / Windsurf / Zed
  3. OpenAI Codex / ChatGPT Desktop
  4. Cline / Continue / Goose / LibreChat
3

Cole a configuração

Copie o trecho para o seu assistente. Cada um precisa de três coisas:

  1. O servidor URL
  2. Authorization: Bearer + seu token
  3. Reinicie seu assistente
05 · Configurações prontas

Escolha seu assistente — a configuração está pronta.

Cada trecho conecta os 4 servidores de uma vez. Basta substituir YOUR_TOKEN pelo token que você criou no passo 1. Se você precisa de apenas um mercado, mantenha somente aquele bloco e exclua o resto.

06 · Segurança

Seu token, sua responsabilidade. As proteções já vêm integradas.

O MCP dá à IA as chaves da sua conta de operações. Por isso nós — e você — acompanhamos os limites juntos.

🔐

O token vive nos cabeçalhos, não no chat

O token Bearer é anexado ao cabeçalho HTTP na conexão assistente ↔ servidor. A IA não o vê no corpo da mensagem e não pode enviá-lo por acidente.

⚠

Não compartilhe. Não faça commit.

O token = acesso à sua conta de operações. Nunca publique em git, capturas de tela, chats ou gists. Se você suspeitar de um vazamento — abra Configurações → Integrações de IA e clique em Revogar ao lado daquele token. Todas as requisições que o usam param de funcionar instantaneamente.

i

Validade e rotação

Você define a validade na criação. Após essa data, o token para de funcionar automaticamente. Recomendamos manter tokens separados por integração — assim você pode revogar apenas o que precisar.

!

Comece com pouco

A IA é poderosa, mas não é infalível. Ela pode escolher o instrumento, o tamanho ou a direção errados. Antes de confiar à IA o dimensionamento — percorra os cenários manualmente e confira suas sugestões duas vezes. A responsabilidade por cada operação é sua.

08 · Perguntas frequentes

FAQ

Preciso instalar algo no meu computador?

Não. Nossos servidores MCP são hospedados por nós — sem repositórios para clonar, sem Python ou venv para instalar. Apenas a URL + token Bearer na configuração do seu assistente de IA.

Um token realmente funciona em todos os 4 servidores?

Sim. Crie um único token Bearer em Settings → AI integrations e use-o no cabeçalho Authorization para os quatro endpoints: digital-options, marginal-cfd, marginal-crypto, marginal-forex.

E se eu quiser conectar apenas um mercado, por exemplo, Forex?

Exclua os outros blocos da configuração — mantenha apenas iq-margin-forex. A IA verá somente as ferramentas daquele servidor e não conseguirá mexer, por exemplo, em opções por acidente.

O que a IA realmente pode fazer pelo MCP?

Cada um dos nossos servidores expõe sua própria lista de ferramentas — você pode ver o conjunto completo logo após conectar, perguntando ao assistente: "Quais ferramentas da IQ Option você tem?". No total, isso é leitura de dados de mercado, portfólio e operações, além de métodos para abrir/fechar posições. Quais métodos estão disponíveis para um token específico é determinado pelo seu escopo (leitura + opcionalmente operar por mercado).

Como garanto que a IA não possa operar?

Ao criar o token em Settings → AI integrations simplesmente não marque nenhuma das quatro caixas "Operar · …". A leitura permanece ativa (não pode ser desativada — a IA precisa dela), e qualquer tentativa de chamar uma ferramenta de operação retornará 403 Forbidden. Esta é a configuração mais segura para casos de uso de pesquisa.

Posso permitir operar em apenas um instrumento?

Sim. Por exemplo, marque apenas Trade · Margin Crypto — a IA poderá abrir/fechar posições em cripto, mas opções, CFD e forex permanecem somente leitura. Útil quando você está testando a configuração da IA em um mercado com o qual se sente confortável.

E se eu quiser remover uma permissão de operar de um token existente?

Abra Settings → AI integrations, encontre o token e desmarque a caixa correspondente (por exemplo, Trade · Margin CFD). A permissão é removida instantaneamente — o servidor começará a retornar 403 Forbidden nas chamadas de operação para aquele mercado. O valor do token em si não muda, então não é preciso mexer na configuração do seu assistente.

E se meu token vazou?

Vá para Settings → AI integrations, encontre o token comprometido e clique em Revogar. Todas as requisições com ele param de funcionar instantaneamente. Crie um novo e atualize a configuração do assistente.

Quais são os limites de tokens e servidores?

Uma conta pode ter vários tokens ativos ao mesmo tempo (visíveis na lista de tokens ativos). Um token funciona em todos os 4 servidores. Se você quiser chaves separadas por integração para facilitar a revogação — crie várias e dê a elas nomes significativos.

Meu assistente não vê as ferramentas após a configuração. E agora?

Checklist: (1) reinicie o assistente completamente, (2) certifique-se de que o token foi copiado sem espaços extras no início/fim, (3) verifique se a URL não tem barra final e corresponde à sua assinatura, (4) certifique-se de que o cabeçalho Authorization começa exatamente com Bearer (observe o espaço), (5) verifique os logs do assistente — geralmente é um erro de sintaxe JSON na configuração.

Qual assistente é o mais fácil de configurar do zero?

Se você não é desenvolvedor — Claude Desktop ou Cherry Studio: ambos têm uma interface que cuida da configuração para você. Se você vive no código — Cursor ou Claude Code parecerão mais naturais.

Usar o MCP é gratuito?

No momento — sim, gratuito para todos os clientes da IQ Option com uma conta ativa. Você paga apenas pela assinatura do seu assistente de IA (Claude Pro / ChatGPT Plus, etc.).
