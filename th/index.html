01 · MCP คืออะไร

หนึ่งโปรโตคอล AI ตัวใดก็ได้สื่อสารกับบัญชีของคุณ

MCP (Model Context Protocol) เป็นมาตรฐานเปิดจาก Anthropic ที่ผู้ช่วย AI ใช้เชื่อมต่อกับระบบภายนอก เราโฮสต์เซิร์ฟเวอร์สำหรับเครื่องมือแต่ละประเภท — คุณเพียงสร้างโทเคนแล้ววางลงในผู้ช่วยของคุณ

⚡

วิเคราะห์ตลาดด้วยภาษาธรรมดา

ถาม AI เกี่ยวกับราคาสด แท่งเทียนย้อนหลัง สเปรด ปริมาณการซื้อขาย — ด้วยคำพูดของคุณเอง ครอบคลุมทั้ง 4 ตลาดพร้อมกัน

📊

ดูพอร์ตของคุณในแชต

สถานะการลงทุนที่เปิดอยู่ คำสั่งที่รอดำเนินการ ประวัติการเทรด มูลค่ารวมของบัญชี — ทั้งหมดผ่านการสนทนาธรรมดา ไม่ต้องสลับหน้าต่าง

🛡️

ควบคุมในระดับโทเคน

โทเคนแต่ละตัวคือการเชื่อมต่อแยกที่มีอายุการใช้งานของตัวเอง ดูโทเคนที่ใช้งานอยู่ทั้งหมดได้ใน Settings → AI integrations และเพิกถอนได้ด้วยคลิกเดียว

03 · สิทธิ์ของโทเคน

อ่านโดยค่าเริ่มต้น เทรด — เลือกเปิดเอง แยกตามแต่ละตลาด

โทเคนแต่ละตัวมีขอบเขตของตัวเอง สิทธิ์อ่านสำหรับเซิร์ฟเวอร์ทั้ง 4 จะเปิดใช้งานอัตโนมัติและปิดไม่ได้ — AI ต้องใช้สิทธิ์นี้เพียงเพื่อ "เห็น" ข้อมูลของคุณ สิทธิ์ในการ เปิดและปิดการเทรด จะมอบให้อย่างชัดเจน — แยกตามแต่ละตลาดเมื่อคุณสร้างโทเคน

🟠 เทรด — แม่นยำ ตามกรณีการใช้งาน

ต้องการให้ AI ช่วยเทรดเฉพาะออปชันใช่ไหม? ติ๊ก Trade · Digital Options เท่านั้น Margin CFD/Crypto/Forex จะยังคงเป็นแบบอ่านอย่างเดียว — แม้ AI พยายาม เซิร์ฟเวอร์ก็จะตอบกลับ 403 Forbidden.

🔵 เปลี่ยนขอบเขตได้โดยไม่ต้องสร้างโทเคนใหม่

ต้องการยกเลิกสิทธิ์เทรดจากโทเคนที่ใช้งานอยู่แล้วใช่ไหม? เปิด Settings → AI integrationsหาโทเคน ยกเลิกการติ๊กตลาดนั้น — สิทธิ์จะถูกยกเลิกทันที ค่าของโทเคนเองไม่เปลี่ยนแปลง จึงไม่ต้องอัปเดตคอนฟิกของผู้ช่วย

04 · วิธีเชื่อมต่อ

สามขั้นตอน ไม่ต้องเขียนโค้ด ไม่ต้องใช้เทอร์มินัล

MCP ของเราทำงานผ่าน streamable-HTTP — ไม่ต้องมีสภาพแวดล้อม Python ไม่ต้องมี virtual venv ไม่ต้องโคลน repo เพียงแค่ URL + โทเคน Bearer ที่คุณสร้างได้ด้วยคลิกเดียว

1

สร้างโทเคน

ในแอปมือถือหรือบนเว็บ:

  1. เปิด Settings → AI integrations →
  2. แตะ + Add integration
  3. ตั้งชื่อโทเคนและกำหนดวันหมดอายุ
  4. ติ๊กตลาดที่อนุญาตให้เทรดได้ (การอ่านเปิดอยู่เสมอ) รายละเอียด — ส่วน 03 ด้านบน
  5. คัดลอกโทเคนทันที — เมื่อปิดหน้าจอแล้วจะไม่แสดงอีก
เปิด AI integrations →
2

เลือกผู้ช่วยของคุณ

รองรับ MCP โดยตรงใน:

  1. Claude Desktop / Claude Code
  2. Cursor / Windsurf / Zed
  3. OpenAI Codex / ChatGPT Desktop
  4. Cline / Continue / Goose / LibreChat
3

วางคอนฟิก

คัดลอกสนิปเป็ตสำหรับผู้ช่วยของคุณ แต่ละตัวต้องการสามสิ่ง:

  1. เซิร์ฟเวอร์ URL
  2. Authorization: Bearer + โทเคนของคุณ
  3. รีสตาร์ต ผู้ช่วยของคุณ
05 · คอนฟิกสำเร็จรูป

เลือกผู้ช่วยของคุณ — คอนฟิกพร้อมแล้ว

สนิปเป็ตแต่ละชุดเชื่อมต่อเซิร์ฟเวอร์ทั้ง 4 พร้อมกัน เพียงแทนที่ YOUR_TOKEN ด้วยโทเคนที่คุณสร้างในขั้นตอนที่ 1 หากต้องการเพียงตลาดเดียว ให้เก็บเฉพาะบล็อกนั้นไว้และลบที่เหลือ

Claude Desktop

ง่าย
แอปเดสก์ท็อปของ Anthropic สำหรับ Mac และ Windows รองรับ MCP โดยตรง แก้ไขไฟล์ JSON ไฟล์เดียว รีสตาร์ต — เสร็จเรียบร้อย
▸ Mac
~/Library/Application Support/Claude/claude_desktop_config.json
▸ Windows
%APPDATA%\Claude\claude_desktop_config.json

Claude Code (CLI)

ง่าย
AI บนเทอร์มินัลของ Anthropic หนึ่งคำสั่งเชลล์ต่อหนึ่งเซิร์ฟเวอร์ ไม่ต้องมีไฟล์คอนฟิก — ทุกอย่างผ่าน CLI
เทอร์มินัล · เพิ่มเซิร์ฟเวอร์ 4 ตัว 
# Digital Options
claude mcp add --transport http iq-digital-options \
  https://digital-options.mcp.iqoption.com \
  --header "Authorization: Bearer YOUR_TOKEN"

# Margin CFD
claude mcp add --transport http iq-margin-cfd \
  https://marginal-cfd.mcp.iqoption.com \
  --header "Authorization: Bearer YOUR_TOKEN"

# Margin Crypto
claude mcp add --transport http iq-margin-crypto \
  https://marginal-crypto.mcp.iqoption.com \
  --header "Authorization: Bearer YOUR_TOKEN"

# Margin Forex
claude mcp add --transport http iq-margin-forex \
  https://marginal-forex.mcp.iqoption.com \
  --header "Authorization: Bearer YOUR_TOKEN"
✓
ตรวจสอบ รัน claude mcp list — คุณควรเห็น 4 บรรทัดที่มีสถานะ ✓ เชื่อมต่อแล้ว

08 · คำถามที่พบบ่อย

คำถามที่พบบ่อย

ฉันต้องติดตั้งอะไรในคอมพิวเตอร์ของฉันไหม?

ไม่ต้อง เซิร์ฟเวอร์ MCP ของเราโฮสต์โดยเรา — ไม่ต้องโคลน repo ไม่ต้องติดตั้ง Python หรือ venv เพียงแค่ใส่ URL + โทเคน Bearer ในคอนฟิกของผู้ช่วย AI ของคุณ

โทเคนเดียวใช้ได้กับเซิร์ฟเวอร์ทั้ง 4 จริงหรือ?

ใช่ สร้างโทเคน Bearer หนึ่งตัวใน Settings → AI integrations แล้วใช้ในเฮดเดอร์ Authorization สำหรับทั้งสี่เอนด์พอยต์: digital-options, marginal-cfd, marginal-crypto, marginal-forex

จะเป็นอย่างไรถ้าฉันต้องการเชื่อมต่อเพียงตลาดเดียว เช่น Forex?

ลบบล็อกอื่น ๆ ออกจากคอนฟิก — เก็บไว้เฉพาะ iq-margin-forex AI จะเห็นเฉพาะเครื่องมือจากเซิร์ฟเวอร์นั้นและจะไม่สามารถแตะต้อง เช่น ออปชัน โดยไม่ตั้งใจได้

จริง ๆ แล้ว AI ทำอะไรได้บ้างผ่าน MCP?

เซิร์ฟเวอร์แต่ละตัวของเราเปิดเผยรายการเครื่องมือของตัวเอง — คุณสามารถดูชุดเต็มได้ทันทีหลังเชื่อมต่อโดยถามผู้ช่วยว่า: "คุณมีเครื่องมือ IQ Option อะไรบ้าง?" โดยรวมแล้วคือการอ่านข้อมูลตลาด พอร์ต และการเทรด รวมถึงเมธอดในการเปิด/ปิดสถานะ เมธอดใดที่ใช้ได้กับโทเคนเฉพาะตัวขึ้นอยู่กับขอบเขตของมัน (อ่าน + เลือกเทรดได้ตามแต่ละตลาด)

ฉันจะทำให้แน่ใจได้อย่างไรว่า AI เทรดไม่ได้?

ตอนสร้างโทเคนใน Settings → AI integrations เพียง อย่าติ๊ก ช่องทำเครื่องหมาย "เทรด · …" ใด ๆ ในสี่ช่องนั้น การอ่านยังคงเปิดอยู่ (ปิดไม่ได้ — AI ต้องใช้) และความพยายามใด ๆ ที่จะเรียกเครื่องมือเทรดจะตอบกลับ 403 Forbidden นี่คือการตั้งค่าที่ปลอดภัยที่สุดสำหรับกรณีการใช้งานเชิงวิจัย

ฉันสามารถอนุญาตให้เทรดเฉพาะเครื่องมือเดียวได้ไหม?

ได้ ตัวอย่างเช่น ติ๊กเฉพาะ Trade · Margin Crypto — AI จะสามารถเปิด/ปิดสถานะคริปโตได้ แต่ออปชัน, CFD และ forex จะยังคงเป็นแบบอ่านอย่างเดียว มีประโยชน์เมื่อคุณกำลังทดสอบการตั้งค่า AI กับตลาดที่คุณคุ้นเคย

จะเป็นอย่างไรถ้าฉันต้องการยกเลิกสิทธิ์เทรดจากโทเคนที่มีอยู่?

เปิด Settings → AI integrationsหาโทเคนแล้วยกเลิกการติ๊กช่องที่เกี่ยวข้อง (เช่น Trade · Margin CFD) สิทธิ์จะถูกยกเลิกทันที — เซิร์ฟเวอร์จะเริ่มตอบกลับ 403 Forbidden สำหรับการเรียกเทรดในตลาดนั้น ค่าของโทเคนเองไม่เปลี่ยนแปลง จึงไม่ต้องแตะต้องคอนฟิกของผู้ช่วย

จะเป็นอย่างไรถ้าโทเคนของฉันรั่วไหล?

ไปที่ Settings → AI integrationsหาโทเคนที่ถูกบุกรุกแล้วกด Revoke คำขอทั้งหมดที่ใช้โทเคนนั้นจะหยุดทำงานทันที สร้างโทเคนใหม่แล้วอัปเดตคอนฟิกของผู้ช่วย

ข้อจำกัดของโทเคนและเซิร์ฟเวอร์มีอะไรบ้าง?

หนึ่งบัญชีสามารถมีโทเคนที่ใช้งานได้หลายตัวพร้อมกัน (ดูได้ในรายการ Active tokens) โทเคนหนึ่งตัวใช้ได้กับเซิร์ฟเวอร์ทั้ง 4 หากต้องการแยกกุญแจตามแต่ละการเชื่อมต่อเพื่อให้เพิกถอนได้ง่ายขึ้น — ให้สร้างหลายตัวและตั้งชื่อที่มีความหมาย

ผู้ช่วยของฉันมองไม่เห็นเครื่องมือหลังตั้งค่า ต้องทำอย่างไร?

รายการตรวจสอบ: (1) รีสตาร์ตผู้ช่วยอย่างสมบูรณ์ (2) ตรวจสอบให้แน่ใจว่าคัดลอกโทเคนมาโดยไม่มีช่องว่างเกินที่ต้น/ท้าย (3) ตรวจสอบว่า URL ไม่มีเครื่องหมายสแลชต่อท้ายและตรงกับการสมัครสมาชิกของคุณ (4) ตรวจสอบให้แน่ใจว่าเฮดเดอร์ Authorization ขึ้นต้นด้วย Bearer พอดี (สังเกตช่องว่าง) (5) ตรวจสอบบันทึกของผู้ช่วย — โดยปกติจะเป็นข้อผิดพลาดไวยากรณ์ JSON ในคอนฟิก

ผู้ช่วยตัวไหนตั้งค่าจากศูนย์ได้ง่ายที่สุด?

หากคุณไม่ใช่นักพัฒนา — Claude Desktop หรือ Cherry Studio: ทั้งคู่มี UI ที่จัดการคอนฟิกให้คุณ หากคุณอยู่กับโค้ดเป็นประจำ — Cursor หรือ Claude Code จะรู้สึกเป็นธรรมชาติกว่า

การใช้ MCP ฟรีไหม?

ในขณะนี้ — ใช่ ฟรีสำหรับลูกค้า IQ Option ทุกคนที่มีบัญชีที่ใช้งานอยู่ คุณจ่ายเฉพาะค่าสมัครสมาชิกผู้ช่วย AI ของคุณเท่านั้น (Claude Pro / ChatGPT Plus ฯลฯ)
